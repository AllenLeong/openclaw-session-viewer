## Features

1. **Agent Grouping** - Sessions grouped by agent with expandable/collapsible headers
2. **Avatar Display** - Agent avatars loaded from workspace configuration
3. **Filter & Search** - Filter by status, agent, or search by session ID
4. **Timeline Navigation** - Sticky headers and thumbnail bar for easy navigation
5. **Message Cards** - Color-coded by role (user/assistant/tool) with markdown support
